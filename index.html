HelloWorld

History

Test programs have always existed in software development, but the tradition of using "Hello, World" as a test message was introduced as an example program in the early 1970s. The C Programming Language "Hello, World" example was taken from a 1974 Bell Laboratories internal memorandum by Brian Kernighan, one of the co-creators of the Unix operating system.

Example

Here is a JavaScript example:


<!DOCTYPE HTML>
<html>
<head>
<script>
alert('Hello, world!');
</script>
</head>
<body>
</body>
</html>
